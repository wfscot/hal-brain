# Project: Gruntweave

Custom CI/CD platform that Scot built during pre-CTO consulting. Foundation for IFC's new cloud-native stack.

## Overview

- Platform infrastructure managed by Nathan Toups
- All new apps (portal, data platform) deploy through Gruntweave
- Repo: gw-system-live (agentic root for all Gruntweave development)
- Supporting repo: gw-agentic (agent tools, rulesync exports)
- Designed for agentic engineering from the outset
- Terragrunt/Terraform for infrastructure, Nx monorepo structure
