# Project: Data Platform

Azure Databricks-based analytics platform (EDH/ALH), replacing analytical workloads currently straining Dynamics.

## Overview

- Built by Nathan Toups and Innocent Musanzikwa
- Data source onboarding tracker in place
- First SFTP pipeline running as of early March 2026
- Zillow FIPS mapping, Python library pattern, 82 automated tests
- Servicing data integration targeted for April 2026

## Purpose

Move analytical and reporting workloads out of Dynamics (operational DB) into a proper analytical platform. Dynamics was never meant for this and it's causing scalability issues.
